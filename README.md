# Node JS Back-End

> Node Js project

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:2020
node index.js
nodemon index.js

```

## API Requests

``` bash
GET localhost:2020/api/v1/details/showall
GET localhost:2020/api/v1/details/show/([0-9])/
DELETE localhost:2020/api/v1/details/remove/([0-9])/
